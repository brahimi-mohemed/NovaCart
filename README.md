# Nova Store

A modern, responsive e-commerce web application built with **React** and **Vite**. Nova Store provides a complete online shopping experience, allowing users to browse products, search and filter the catalog, manage a shopping cart, and complete a simulated checkout process.

## Live Demo

nova-cart-website.vercel.app


## Features

* Responsive layout for desktop and mobile devices
* Browse a catalog of products
* Product details page with images, ratings, stock status, and descriptions
* Search products by name
* Filter products by category
* Sort products by different criteria
* Reset all search and filter options
* Client-side pagination
* Shopping cart with quantity management
* Add and remove products from the cart
* Cart badge displaying the number of items
* Toast notifications when products are added to the cart
* Checkout page with shipping information form
* Order confirmation page
* About page
* Custom 404 Not Found page
* Persistent cart using Local Storage

## Tech Stack

* React
* React Router

## Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/nova-store.git
```

Navigate into the project folder:

```bash
cd nova-store
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Create a production build:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
